Product Description

Turtle Up Tracker is a mobile app that enables users to track turtles in real time and to support the Turtle Up organization. It provides detailed information about turtles, options to purchase Turtle Up products, methods to donate to the organization, and access to additional resources about Turtle Up.


App Overview




Navigation

  • Home: Explore and connect to specific turtles
  • Map: Real time map visualization simulation
  • Store: Shopping and donation experience
  • Info: Direct links to company sites/social media accounts
  • Admin: Admin user capabilities to modify/delete/create turtles


Architecture Design




Technical Details

  • Frontend: React Native, Expo, TypeScript
  • Backend: Go, Gin
  • Database: FireStore
  • Map: Google Maps API
  • Updates: GCP Run Functions, Pub/Sub, Scheduler Service
  • Payment: Stripe
  • Newsletter: Mailchimp
