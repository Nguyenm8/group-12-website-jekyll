Product Description

Turtle Up Tracker is a mobile app built with React Native and Firebase. It enables users to track turtles in real time while providing ways to support the Turtle Up organization. The app offers information on turtles, options to purchase Turtle Up products, donation capabilities, and access to additional resources about the organization.
